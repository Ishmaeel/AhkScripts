# AutoHotkey Scripts Collection

A collection of AutoHotkey scripts.

## Installation

Download and install AutoHotkey from [https://www.autohotkey.com/](https://www.autohotkey.com/)

### Hotkey Modifiers
- `^` = Ctrl
- `!` = Alt  
- `+` = Shift
- `#` = Windows key
